# Art Nouveau Lamp

This was my approach for the #ElectromakerKitsCompetition at [MyMiniFactory](https://www.myminifactory.com/object/3d-print-art-nouveau-mood-lamp-118082)

A Wemos D1 mini (ESP8266) controls a ring of 16 WS2812 RGB-LEDs. The "Cop Alarm" effect is used to signal that HomeAssistant detected a critical state.
